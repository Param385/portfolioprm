A personal portfolio website showcasing professional experience, cybersecurity and IT skills, projects, and contact information.

## Overview

This is my personal portfolio website built using HTML, CSS, and JavaScript.

The site showcases:

- ✅ Professional experience  
- ✅ Cybersecurity and IT skills  
- ✅ Projects  
- ✅ Contact information

## Technologies & Tools

**Languages:** HTML5, CSS3, and vanilla JavaScript — no frameworks or build tools required.  

**Fonts:** Google Fonts — Syne (headings), DM Mono (monospace/labels), DM Sans (body text).  

**Form Backend:** Formspree — handles contact form submissions and routes them to email without a backend server.  

**CSS Techniques:**  
CSS custom properties (variables), CSS Grid, Flexbox, backdrop-filter, IntersectionObserver-driven scroll animations, and a procedural noise texture via an inline SVG feTurbulence filter.

**JavaScript Features:**  
Plain browser-native JS — IntersectionObserver for scroll-reveal animations, the Fetch API for async form submission to Formspree, and a scroll event listener for active nav highlight and back-to-top button.
